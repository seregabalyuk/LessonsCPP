# 1 задание: Обертка над функцией

В питоне, можно создавать обертки функций.
Но в с++, напрямую так сделать нельзя.

Поэтому у с++ есть свои инструменты.
Например использовать класс, как функцию.

Пример:
```c++
struct Sum {
  // мы переопределяем оператор круглые скобочки для нашего класса.
  // благодаря ему мы можем использовать класс, как функцию
   
  int operator()(int a, int b) {
  //^            ^------^-- параметры
  //|--- возвращаемый тип
    return a + b;
  }

  int operator()(int a, int b, int c) {
    return a + b + c;
  }
};

int main() {
  Sum sum;
  int f = sum(0, 1);
  int g = sum(1, 2, 3);
}
```

Задание: 
Реализовать особую обертку над функцией.

Для этого использовать такой шаблон кода:

```cpp
class Wrap {
  float(*)(float) function_1; // указатель на функцию
  //^      ^ -- параметры
  //|--- возращаемый тип
  float(*)(float) function_2; // указатель на функцию
  /*
  функции, конструкторы, операторы, деструкторы класса
  */
};
/*
Для вызова используй круглые скобки
function_1(0.5f);
*/
```

Код должен, сначала вызвать одну функцию, потом результат передать в другую.

При работе с углами в с++ используются  [радианы](https://ru.wikipedia.org/wiki/Радиан), а не градусы.
Но благодоря этой обертке мы можем работать с этими функциями как будто работаем с градусами.
Пример использования:

```cpp
#include <cmath>

float Radian2Degree(float in) {
  return in * (180.f / M_PI);
}

float Degree2Radian(float in) {
  return in * (M_PI / 180.f);
}

int main() {
  const Wrap wcos(Degree2Radian, std::cos);
  const Wrap wsin(Degree2Radian, std::sin);
  const Wrap watan(std::atan, Radian2Degree)
  std::cout << wcos(90);
}
```

# 2 задание: Оператор []

Обычно оператор [] применяют для обращения к элементу массива. Вот примерный код:

```cpp
struct MyContainer {
  Vaue operator [](/* ключ */) {
    return /*элемент массива*/;
  }
};
```

Задание:
Реализовать массив.
Он должен добавлять брать память из кучи, удалять ее.


Пример использования:
```cpp
int main() {
  Massive a; // память не должна выделяться
  Massive b(10); // память выделяется на 10 элементов, каждый равен 0
  Massive с(11, 5); // память выделяется на 11 элементов, каждый равен 0
  const Massive d(b); // копирует массив.
  
  // функции
  b.clear(); // возвращает память в кучу. Размер 0;
  c.resize(10); // делает новый размер, удаляет старый массив, выделяет новый. Заполняет его 0
  c.resize(99, 77); // делает новый размер, удаляет старый массив, выделяет новый. Заполняет его 77
  d.size(); // возращает длину массива
  d.empty(); // если массив пуст возращает true, иначе false

  std::cout << d[4]; // есть обращение по номеру
  c[4] = 100; // можно менять по номеру.

  a = b; // есть оператор присваивания.
}
```

Подробнее про оператор копирования
```cpp
struct Class {
  // оператор копирования
  // Мы переопределяем =
  // Важно! Не путать с конструктором копирования.
  // Разница в том, что при операторе копирования, экземляр класса уже создан.
  // А при конструкторе копирования объект не создан ещё.
  Class& operator=(const Class& other) {
    /* копирование */
    return *this;
  }
};
```

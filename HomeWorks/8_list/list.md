# List

## Задача 1

Нужно реализовать функцию отчиски списка.

```cpp
template<class T, class Func>
void clean(std::list<T>& in, const Func& func) {
  // func это условие по которому удаляется элемент
  // Так как это шаблон, он может быть функцией.
  for (auto iter = in.begin(); iter != in.end(); ++ iter) {
    if (func(*iter)) {
      /*
      Твой код.
      Нужно удалить элемент по итератору. 
      При этом ничего не должно сломаться :3
      */
    }
  }
}
```

Примеры использования:

```cpp
// функция которая указывает какие элементы удалять
// тут удаляются отрицательные числа
bool checker(int num) { return num < 0; }

int main() {
  std::list<int> mylist = {-7, 1, -1, 2, 4, 6, -19, -8, -77, 7, 9, 0, -5};
  clean(mylist, checker);
  for (auto value: mylist) {
    std::cout << value << ' ';
  }
  std::cout << '\n';
}
```
Вывод должен быть таким:
```
1 2 4 6 7 9 0
```

## Задача 2

Есть метод сжатия данных.
Допустим у тебя последовательность английских строчных букв:
```
helloooooohellooooworlddddd
```
Тогда её можно сжать, при помощи опреаций повторения.
```
2(hell6(o))worl5(d)
```
Если у нас есть повтояющиеся части, мы их закрываем скобочками и в начале пишем количество повторений.
Заметим, что внутри скобочек может быть ещё скобочки.

Но мы не будем реализовывать шифратор. Мы будем реализовывать дешифратор. 

На вход программе подоётся сжатая строка, нужно вывести исходную строку.

Реализуй через `list` и `vector`.

А потом подставь такие значения:
```
500(200(a)100(b))
```
И посмотри, кто быстрее работает.

### Идея решения

Идешь итеративно по строке, и если видишь цифры, запоминаешь позицию и много раз вставляешь нужный участок.

# union

Реализуй класс треугольник.

Он должен работать на таком коде.

```cpp
int main () {
  Triangle tr;
  tr.A.x = 10;
  tr.B.y = 0;
  tr.C;
  tr.array[1] = Point(1, 2);
  // тут мы можем по разному обращаться к точкам треугольника
  // через буквы
  // и через массив
  // При этом они не должны обращаться к разным участкам памяти.
}
```

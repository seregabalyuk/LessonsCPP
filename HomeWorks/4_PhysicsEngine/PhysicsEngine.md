# Engine

## Класс Body

Углубимся в физику.

Для простоты реализации, тела нельзя будет вращать.

У каждого тела, которое ни с чем не взаимодествует, есть несколько параметров.
1) Масса - обычно не меняется
2) Позиция
3) Скорость

А теперь к формулам:
1) Позиция и скорость меняются со временем.
    Вот по такому закону:
    $$p(t + dt) = v \cdot dt + p(t)$$
    где $p$ позиция, $v$ скорость, $t$ время, $dt$ изменение времени.

2) Так же если к телу прилогается **ускорение**, то скорость меняется по такому закону:
    $$v(t + dt) = a \cdot dt + v(t)$$
    где $v$ скорость, $a$ сила, $t$ время, $dt$ изменение времени.

3) Но мы не будем работать с ускорением, так как оно зависит от конкретного тела. 
    Мы будем работать с силой.
    $$f = m \cdot a$$
    где $f$ - сила, $m$ - масса тела, $a$ - ускорение тела.

    Есть один замечательный закон:
    Сила действия, равна силе противодействия.

    Поэтому, если объекты сталкиваются, на них действуют одинаковые силы, но направленные в противоположные стороны.

Так же нужно заметить, что движки не непрерывные, а расчитывают физику кадрами.

То есть между 2 мя кадрами прошло $dt$ времени. Сначало мы рассматриваем все силы, изменяем все скорости, и в конце изменяем позицию.

Реализуй класс Body:
```cpp
struct Body {
  /// переменные класса
  void applyForce(const Vector& force); // действие силы на тело (меняется скорость)
  void update(); // Обновление позиции
};
```

Для проверки, реализуй программу на sfml. 
Должна быть пушка, которая стреляет ядрами.
Ядра вылетают с заданной скоростью.
На ядра действует сила гравитации (константная сила).

Пример кода на sfml:
```cpp

#include <SFML/Graphics.hpp>


void draw(sf::RenderWindow& window, float deltaTime)
{
    sf::Vector2i mousePos = sf::Mouse::getPosition(window);
    // Получаем координаты мыши
}

int main()
{
    sf::RenderWindow window(sf::VideoMode(800, 600), "SFML Window");
    
    sf::Clock clock;
    
    while (window.isOpen())
    {
        sf::Event event;
        while (window.pollEvent(event))
        {
            if (event.type == sf::Event::Closed)
                window.close();
        }
        
        float deltaTime = clock.restart().asSeconds();
        
        
        window.clear(sf::Color::Black);
        
        
        draw(window, deltaTime);
        
        window.display();
    }
    
    return 0;
}
```
